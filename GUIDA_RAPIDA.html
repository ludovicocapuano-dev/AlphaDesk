Fase 1 — Configura SSH dal tuo Mac

1 Genera una chiave SSH (se non ne hai una)

Apri il Terminale sul tuo Mac e digita:

ls ~/.ssh/id_ed25519.pub 2>/dev/null || ssh-keygen -t ed25519 -C "ludovicocapuano@gmail.com"

Premi Enter a tutte le domande (password vuota va bene per ora).

2 Copia la chiave sul VPS

Sostituisci IL_TUO_IP con l'IP del VPS Hetzner:

ssh-copy-id root@IL_TUO_IP

Ti chiederà la password del VPS (la trovi nell'email di Hetzner). Dopo questo, non servirà più la password.

Fase 2 — Setup automatico del VPS

3 Connettiti al VPS e lancia il setup

ssh root@IL_TUO_IP

Una volta dentro, incolla tutto questo (è il contenuto di SETUP_VPS.sh):

bash <(cat <<'SCRIPT'
# ... il contenuto di SETUP_VPS.sh ...
# Scaricalo dalla cartella alphadesk/ e copialo qui
SCRIPT
)

Oppure, metodo più semplice:

# Dal tuo Mac (NON dal VPS):
scp SETUP_VPS.sh root@IL_TUO_IP:/tmp/
ssh root@IL_TUO_IP "chmod +x /tmp/SETUP_VPS.sh && /tmp/SETUP_VPS.sh"

Ci vogliono ~3-5 minuti. Installa Python, PyTorch, crea l'utente, configura il firewall.

Fase 3 — Upload dei file

4 Carica i file Python sul VPS

Dal tuo Mac, nella cartella alphadesk/:

chmod +x UPLOAD_TO_VPS.sh
./UPLOAD_TO_VPS.sh IL_TUO_IP

Trasferisce tutti i 30+ file Python nella struttura corretta.

Fase 4 — Configura le chiavi (UNICA COSA MANUALE)

5 Inserisci le API key di eToro

ssh root@IL_TUO_IP
nano /home/alphadesk/app/.env

Modifica queste due righe con le tue chiavi reali:

ETORO_USER_KEY=la_tua_chiave_qui
ETORO_API_KEY=la_tua_chiave_qui

Salva: Ctrl+X → Y → Enter

Lascia ETORO_ENV=Demo! Non cambiare in "Real" finché non hai testato per almeno 2-3 mesi.

Fase 5 — Avvia!

6 Avvia AlphaDesk

# Abilita avvio automatico al boot
systemctl enable alphadesk

# Avvia ora
systemctl start alphadesk

# Guarda i log in tempo reale
journalctl -u alphadesk -f
Se vedi AlphaDesk v2 initializing (Self-Learning Ensemble) nei log, è tutto funzionante!
